# Project Title

Ecommerce Website

## Built With

* CSS
* ReactJS
* Commerce JS
* Stripe

## Getting Started
First, install the packages
```bash
npm i
```
Second, put your api keys in the .env file

Third, run the development server:

```bash
npm run dev
# or
yarn dev
```
